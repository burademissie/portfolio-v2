# Image Assets Guide

## Profile Photo
Place your profile photo at:
- `/public/profile-photo.jpg`

Recommended size: 800x1000px (portrait orientation)
Format: JPG or PNG

## Project Screenshots
Place your project landing page screenshots at:
- `/public/projects/ecommerce.jpg` - E-Commerce Platform
- `/public/projects/taskmanager.jpg` - Task Management App
- `/public/projects/dashboard.jpg` - Social Media Dashboard
- `/public/projects/weather.jpg` - Weather Forecast App
- `/public/projects/music.jpg` - Music Player
- `/public/projects/portfolio.jpg` - Portfolio Website

Recommended size: 1200x800px (landscape orientation)
Format: JPG or PNG

## Tips
- Use high-quality screenshots of your actual project landing pages
- Ensure images are optimized for web (compressed but still clear)
- The portfolio will show gradient placeholders if images are not found
- You can use tools like TinyPNG or ImageOptim to compress images
